# JSONエディタ

シンプルさを重視してGeminiCLIに作成してもらった、Jsonファイル作成、表示、編集用GUIツールです。ゲームプランナーのような非プログラマーでも安全に利用、レベルデザインができることを意図しています。

このツールは、JSON構造を分かりやすいツリー形式で表示し、データ型に応じて適切な編集ウィジェットを動的に生成します。意図しないデータの変更を防ぐ「構造ロック」モードを搭載しており、レベルデザインのパラメータ調整のようなタスクに最適です。

## 主な機能

- **ツリービュー:** JSON構造全体を、直感的な階層ツリーで視覚化します。
- **動的エディタ:** データ型ごとに最適な入力フィールドを自動で提供します。
    - `string`, `number`: テキスト入力
    - `boolean`: チェックボックス
- **構造ロックモード:** 有効にすると以下の動作になる安全機能です。
    - キーや要素の追加・削除を無効化します。
    - 値を編集する際、元のデータ型を強制します。
- **安全な配列・オブジェクト編集:**
    - オブジェクトの配列に新しい要素を追加する際、最後の要素の構造を複製して追加します。
    - （ロック解除時）オブジェクトに新しいキーと値のペアを追加します。
- **ファイル操作:** JSONファイルの読み込み、保存、新規作成に対応しています。
- **モダンなUI:** CustomTkinterで構築されており、システムのテーマに準拠したクリーンでモダンな外観です。

## インストール方法

1.  **リポジトリをクローンします:**
    ```bash
    git clone <your-repository-url>
    cd json_editor_app
    ```

2.  **仮想環境のセットアップ（推奨）:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Windowsの場合は `venv\Scripts\activate` を実行
    ```

3.  **依存ライブラリをインストールします:**
    ```bash
    pip install -r requirements.txt
    ```

## 使い方

`json_editor_app` ディレクトリからアプリケーションを実行します。
```bash
python main.py
```

- **Load File:** ファイルダイアログを開き、`.json` ファイルを選択して読み込みます。
- **New File:** サンプルのJSON構造で新規に開始します。
- **Save:** 変更を保存します。読み込み済みのファイルを上書きするか、新規ファイルの場合は保存先を尋ねます。
- **Lock Structure:** JSONの構造変更を防ぎ、厳密なデータ型を強制する安全モードを切り替えます。
